Que és una variable?

  • Una variable és una manera de guardar dades i de nomenar-les perquè és puguin fer servir més tard.

Per què fem servir les variables?

  • Per a evitar repetir el nostre codi.
  • Perquè podrem canviar valors fàcilment mentre que s'està executant el programa.

Com és crea una variable? var

En crear una variable, tu li poses un nom, el tipus de dades i un valor. També declares o llistes les variables en la part superior del teu esbós.

  • Tipus de dada: paraules, imatges, tipografia.
  • Nom: el que decideixes denominar a la variable (tria un nombre que sigui informatiu i clar).
  • Valor: un nombre, paraula, imatge, etc.

Que és un operador?

  • Els símbols com +, -, i * són operadors .
  • Quan és col·loquen entre dos valors, poden crear una expressió .

  • + suma.
  • - resta.
  • * multiplicació.
  • / divisió.
  • = assignació.

Operadors relacionals

  • > major que.
  • < menor que.
  • >= major que o igual.
  • <= menor que o igual.
  • == igual a.
  • != no igual a.
  • || o.
  • ! no.

Recorde l'ordre d'operacions per PRECEDENCIA D' OPERADORS (PEMDAS): P aréntesis, E xponentes, M ultiplicació, D ivisió, Suma/ A fegir, Resta/ S ustracció

For loops

Hi ha 4 parts en un for loop:

  • La inicialització: declara una nova variable (en general és fa servir la lletra i ).
  • La prova: avalua el valor d'aquesta variable.
  • Les declaracions: el codi.
  • L'actualització: canvis en el valor d'una variable.

Inicialització, prova i actualització treballen en conjunt per a controlar quantes vegades s'executaran les declaracions o codi.

Loop

Exemple 2.3

              
                  function setup() {
                    createCanvas(480,120);
                    background (180,255,255);
                  }
                  
                  function draw() {
                    strokeWeight (8)
                    for( var i = 20; i<=260; i +=40 ){
                    stroke (0,i,0);
                    line (i,40,i,80); //aquesta línea és continuarà dibuixant
                    //sempre i quan "i" sigui menor o igual a 260
                    }
                  }

Repassem el codi anterior

  • La inicialització és var i = 20.
  • La prova és i <= 260.
  • I l'actualització és i += 40.
  • Llavors, per a i = 20 , agregarà 40 a i , per a i = 60 , agregarà 40 a i , i així successivament fins que i sigui major que 260.

Nested for loop

Exemple 2.5

També podem tenir un o més for loop endins d'un for loop . Això és denomina for loop anidat.

            
                // launch your canvas
                function setup() {
                  // put setup code here
                  // init your variables
                  createCanvas(displayWidth, displayHeight);
                  // bg color
                  background(0);
                }
                
                // loop for your draw/animation
                function draw() {
                  // put drawing code here
                  strokeWeight(8)
                  for (var i = 20; i<=displayWidth; i +=40){
                    for  (var j = 20; j<=displayHeight; j +=40) {
                      stroke(j/2,i/2,0);
                      line(i, j+20,i,j+40);
                    }
                  }
                }

Peer programming

La programació en parella és quan dos developers fan servir un sol ordinador. Una persona actua com el Pilot , i l'altra com el Copilot.

  • Pilot
    • Controla el ratolí i el teclat.
    • S'encarrega dels detalls de codificació.
  • Copilot
    • Comparteix el ratolí.
    • Pensa en el que el codi ha de fer en major escala.
    • Revisa que no hi hagi errors o paraules/caràcters mal escrits.

Funcions

  • Una funció li ensenya al programa una nova tasca. És una llista d'instruccions que és guarden amb un nom per a usar-se després. (Com una recepta de cuina).
  • Nosaltres fem servir les funcions per a crear codi manipulable i reutilitzable . Ens estalvia temps!.
  • Per a declarar una funció, comences amb la paraula function . Li poses un nom, igual que com ho faries amb una variable.
  • El nom sempre ha de començar amb una lletra . Les majúscules i minúscules importen! Aquest és un exemple de com una funció de nom target és definida i dita diverses vegades.

Funció Random

  • La funció random()o aleatori (a l'atzar) et dóna un nombre punt flotant aleatori. (Recorda, un nombre punt flotant és sol un nombre decimal).
  • La funció random()pot tenir paràmetres. Els paràmetres són trossos d'informació extra que ajuden a definir una funció.
              
                  random( [min],[max] )
                  rect( random([20],[400]), 20, 20, 20 )

*min i max actuen com els límits inferiors i superiors de la teva funció. Pots deixar els paràmetres buits per a qualsevol nombre aleatori; o pots decidir escriure un valor min, max o tots dos.

Condicional

  • if.
  • then.
  • else.
                
                    if ( weather == ‘nice’ ) {
                        beachCode()
                        } else if ( weather == ‘very nice’ ) {
                        eatIceCream()
                        } else {
                        roomCode()
                      }

En pseucodi seria: En cas de (If) que el clima (weather) sigui bó (nice), aniré a la platja a codificar (beachCode) - si no (else), em quedaré a la meva habitació a codificar (roomCode).

Rotació

Exemple 4.1

La funció rotate() rota el sistema de coordenades per a què les formes és puguin dibuixar en angle. Té un paràmetre que fixa la quantitat de rotació. La quantitat és mesura en unitats de radians o radiants.

                
                    // rotate(PI/8) Usando radianes, el valor dels angles 
                    // s'expressa en relació al valor matemátic de pi (𝜋).
                    function setup() {
                        createCanvas(500,500);
                        stroke(255);
                        background(0);
                      }
                    function draw() {
                      var i = 250
                      var j = 250
                      fill(random(255),0,random(255));
                      rotate (radians(random(360)));
                      rect(0,0,20,20);
                    }

Translate

Exemple 4.2

La funció translate() mou l'origen de la pantalla (0,0) a un altre lloc. Té dos paràmetres (la coordenada x, i la coordenada y).

                
                    //translate(x, y);
                    function setup() {
                      createCanvas(500,500);
                      stroke(255);
                      background(0);
                    }

                    function draw() {
                      var i = 250
                      var j = 250
                      // translate();
                      translate (i,j);
                      fill(random(255),0,random(255));
                      rotate(radians(random(360)));
                      rect(0,0,20, 20);
                    }
